Задание 2, вариант 1
Цель работы: визуализировать изменение уровня углекислого газа в атмосфере за период с 1958 по 1980 годы на основе набора данных co2 из библиотеки statsmodels.Разработать программу с использованием библиотек statsmodels и matplotlib для загрузки данных временных рядов из различных наборов данных и построения графиков их динамики за указанные промежутки времени.
Задачи:
1. Загрузить набор данных co2 из библиотеки statsmodels.
2. Преобразовать индекс данных в отдельный столбец 'date' и установить этот столбец как индекс.
3. Создать столбец 'year', извлекая год из столбца 'date'.
4. Отфильтровать данные, оставив только записи, соответствующие заданному диапазону лет.
5. Построить график, отображающий динамику уровня CO2 за указанный период.
Ход работы:
1. Необходимо установить библиотеку matplotlib, которая используется для построения графиков.
2. Импортируются библиотеки: pandas для работы с данными, statsmodels для загрузки данных.
3. Создаются переменные для работы:
dataset_name = 'co2':  указывается название набора данных, который будет загружен - о концентрации углекислого газа (co2).
start_year = 1958:  начальный год для отображения данных на графике.
end_year = 1980:  конечный год для отображения данных на графике.
4. data = sm.datasets.co2.load_pandas(): вызывается функция load_pandas() из statsmodels для загрузки набора данных co2.  Функция возвращает объект, содержащий данные.
df = data.data:  из объекта data извлекается таблица с данными и присваивается переменной.
5. df['year'] = df.index.year:  создается новый столбец year, содержащий год из индекса date.
df = df[(df['year'] >= start_year) & (df['year'] <= end_year)]:  фильтроваие данных.  Оставляются только строки, где год находится в диапазоне от начального до конечного года.
6. Построение графика: используется библиотека matplotlib.pyplot для создания графика. На графике отображается зависимость уровня CO2 от года. Добавляются метки осей, заголовок, легенда и сетка.
Библиотеки: pandas, statsmodels, matplotlib. 
Алгоритмы: Загрузка данных, преобразование индекса, построение графика.
Выводы:
Разработанный код строит график динамики временных рядов с 1958 по 1980 года.
