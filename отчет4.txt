Задание 2
Цель работы: Разработать программу с использованием Biopython для чтения  файла формата GenBank, извлечения последовательностей CDS, вычисления GC-состава каждой последовательности и вывода информации о последовательностях в порядке возрастания их GC-состава.
Задачи: 
1. Выполнить поиск в базе данных NCBI для видов Brassica oleracea и Solanum lycopersicum.
2. Выбрать по пять записей для каждого вида.
3. Использовать буфер обмена NCBI для временного хранения выбранных записей. 
4. Экспортировать все записи из буфера обмена в один файл формата GenBank 
5. Разработать функцию для чтения файла GenBank, используя Biopython. 
6. Извлечь последовательности CDS из каждой записи. 
7. Разработать функцию для вычисления GC-состава каждой последовательности CDS. 
8. Отсортировать последовательности CDS по возрастанию GC-состава. 
9. Вывести информацию о каждой последовательности CDS.
Алгоритм:
1. Итерация по записям и признакам 
 Используется двойной цикл for, чтобы пройтись по всем записям в GenBank файле и по всем признакам внутри каждой записи.
2. Фильтрация по типу признака
Внутри цикла происходит проверка if feature.type == "CDS", то есть фильтрация по условию.
3. Извлечение и преобразование последовательности
feature.extract(record.seq) — извлекает последовательность ДНК, соответствующую признаку CDS. Затем последовательность преобразуется в строку и переводится в верхний регистр.
4. Вычисление GC-содержания
5. Формирование кортежей и добавление в список
6. Сортировка списка entries

1. Импортируем модуль SeqIO из библиотеки Biopython для работы с файлами GenBank.
2.  Создание переменной  file_path, которая содержит путь к файлу.
3.  Цикл for record in SeqIO.parse(file_path, "genbank") перебирает все записи в файле.
4.  Внутренний цикл for feature in record.features перебирает все признаки (features) в каждой записи.
5.  Условие if feature.type == "CDS" проверяет, является ли текущий признак кодирующей областью (CDS).
6.  seq = str(feature.extract(record.seq)).upper() извлекает последовательность CDS из признака, преобразует ее в строку и переводит все символы в верхний регистр. 
7.  gc_content = (seq.count('G') + seq.count('C')) / len(seq) if seq else 0.0 вычисляет GC-состав последовательности.
8.  desc = record.description or record.id получает описание записи. Если описание отсутствует, используется ID записи.
9.  entries.append((gc_content, f"{record.id}: {desc}, GC = {gc_content}")) добавляет кортеж с GC-составом и строкой информации о последовательности в список entries.
10. entries.sort() сортирует список entries по возрастанию GC-состава.
11. Цикл for gc, entry in entries перебирает отсортированный список и выводит информацию о каждой последовательности в заданном формате.
Инструменты: веб-браузер, сайт NCBI (https://www.ncbi.nlm.nih.gov/), буфер обмена NCB, текстовый редактор, Biopython.
